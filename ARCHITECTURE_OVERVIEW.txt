┌─────────────────────────────────────────────────────────────────────────────┐
│ DATA FLOW - POST CREATION LIFECYCLE                                         │
└─────────────────────────────────────────────────────────────────────────────┘

User creates post
        ↓
completeUpload() triggered
        ↓
AREA TAG DETECTION (areaNameDatabase)
├─ Priority 1: Real US locations (50km threshold)
├─ Priority 2: POI (< 500m threshold)
└─ Priority 3: City fallback
        ↓
Validate currentWiFiNetwork (not null)
        ↓
Upload photo to Firebase Storage (if any)
        ↓
Create postData object
├─ id, content, photoURL, mediaType
├─ networkId: currentWiFiNetwork
├─ county: areaTag (from detection above)
├─ timestamp, likes, dislikes
└─ customTag, linkURL
        ↓
Save to Firebase
└─ networks/shared-network-1/posts/{postId}
        ↓
Track in postRecommendationSystem (ML)
        ↓
Track in postTrackingSystem (Firebase)
└─ user-posts/{userId}/shared-network-1/{postId}
        ↓
Firebase listener triggers
        ↓
gridContent['shared-network-1'] = [posts...]
        ↓
renderGrid()
        ↓
Post visible to user ✓

┌─────────────────────────────────────────────────────────────────────────────┐
│ KEY VARIABLES & STATE                                                       │
└─────────────────────────────────────────────────────────────────────────────┘

currentUserId
  Type: string (Firebase UID)
  Set by: onAuthStateChanged() when user authenticates
  Used by: All tracking systems, voting, post creation
  Example: "abc123xyz..."

currentWiFiNetwork
  Type: string
  Default: "shared-network-1"
  Set by: detectWiFiNetwork() after auth
  Used by: Firebase paths, grid display, post filtering

userLocation
  Type: {lat: number, lon: number}
  Set by: Geolocation API in detectUserGrid()
  Used by: Area tag detection, POI queries

gridContent
  Type: {[networkId]: [post_objects]}
  Structure: { "shared-network-1": [{post}, {post}, ...] }
  Updated by: Firebase listener
  Used by: Grid display, post lookup

poiDatabase
  Type: Array of POI objects
  Loaded from: poi_database_expanded.json
  Updated by: poiManagementSystem.addPOI()
  Length: 7,000+

areaNameDatabase.areas
  Type: Map { "lat_lon" => area_object }
  Size: 20+ pre-loaded
  Updated by: areaNameDatabase.addArea()
